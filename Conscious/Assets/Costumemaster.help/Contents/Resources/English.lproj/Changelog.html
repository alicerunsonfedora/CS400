What's New in The Costumemaster

The latest feature update of The Costumemaster, v1.2.0, includes great new features and improvements.

  • Adds the AI Simulator Tool to run AI simulations with different agents and levels*.
  • Changes the target radius for exit doors to make it easier to go to the next level.
  • Adds improvements to starting and resuming games using the menu bar.
  • Adds a new Help Book to the app.

*The AI Simulator Tool requires a Mac running macOS 10.15 Catalina or greater.
